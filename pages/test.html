How to prove it to yourself

Edit this file (or any page) in the repo, commit, wait a few minutes for GitHub's cache to refresh, then relaunch the app. Whatever you typed shows up here — same as it will for Today's Message and events going forward.

Once you've seen it work, this tab and page can be removed by deleting the "test" entry from app-config.json and this file.
